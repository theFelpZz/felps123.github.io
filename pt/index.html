Projeto - Bigby Wolf

Bigby Wolf é um bot para WhatsApp completo, ele é baseado em um personagem de um game/comics que gosto muito, Bigby Wolf dos quadrinhos “Fables”. (Ah e recomendo muito jogar o game viu, é incrível!).

Para usar o bot vá para este número e digite !help (principal) ou !ajuda.

Comandos

Só um pequeno resumo de todas as funções, você pode ver todas enviando !help para o bot.

!help/!ajuda • te manda uma lista de todos os comandos.
!sticker • converte qualquer imagem/vídeo ou gif em uma figurinha!
!image termo • pesquisa de imagens.
!search termo • pesquisa no Google.
!ris • busca de imagem reversa.
!mp4 termo • baixa vídeos do YouTube.
!mp3 termo • baixa qualquer música do YouTube.
!lyrics nome-da-musica • encontra as letras de qualquer música!
